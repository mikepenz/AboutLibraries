⚠️ This page using experimental Kotlin/Wasm which requires enabling experimental features in your browser.
  • For Chrome 110 or newer:
    1. Open chrome://flags/#enable-webassembly-garbage-collection in the browser.
    2. Enable WebAssembly Garbage Collection.
    3. Relaunch the browser.
    or with Chrome 109 or newer, run the browser with the --js-flags=--experimental-wasm-gc command line argument.
  • For Edge 109 or newer: run the browser with the --js-flags=--experimental-wasm-gc command line argument.
  • For Firefox Nightly 112 or newer:
    1. Open about:config in the browser.
    2. Enable javascript.options.wasm_function_references and javascript.options.wasm_gc.
    3. Refresh this page.
